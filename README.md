# Kibsons JS — React + Vite

Plain JavaScript + React port of the Kibsons-inspired storefront (no TypeScript, no TanStack).

## Setup

```bash
npm install
npm run dev
```

Then open http://localhost:5173

## Build

```bash
npm run build
npm run preview
```

## Stack
- React 18
- Vite 5
- Tailwind CSS v4
- lucide-react icons
